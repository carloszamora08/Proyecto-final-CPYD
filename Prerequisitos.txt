Prerequisitos
- Tener Podman (WSL o PodmanDesktop)
- Crear un network para podman nombre: TORNEONFL

Necesitamos para compartir:
- Contenedor de ActiveMQ
. podman run -d --replace --name artemis --network TORNEONFL -p 61616:61616 -p 8161:8161 -p 5672:5672 -m 256m  apache/activemq-classic:6.1.7

- Contenedor de la db
. podman run -d --replace --name=tournament_db --network TORNEONFL -e POSTGRES_PASSWORD=password -p 5432:5432 postgres:17.6-alpine3.22
. podman exec -i tournament_db psql -U postgres -d postgres < database/db_script.sql
nota: tienen que tener db_script.sql accesible en el disco para poderlo ejecutar

- Contenedor del consumer
. podman build -f tournament_consumer/Containerfile -t tournament_consumer
. podman run --replace -d --network TORNEONFL --name tournament_consumer tournament_consumer
nota: tienen que tener Containerfile de consumer accesible en el disco para poderlo ejecutar

- Contenedor del servicio
. podman build -f tournament_services/Containerfile -t tournament_services
. podman run --replace -d --network TORNEONFL --name tournament_services -p 8081:8080 tournament_services
nota: tienen que tener Containerfile de services accesible en el disco para poderlo ejecutar
